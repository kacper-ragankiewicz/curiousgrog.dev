# Installing node_modules

To install node modules:
```
    $ sudo yarn install
```
To upgrade modules:
```
    $ sudo yarn upgrade
```
<br>

# Adding modules //your env must be active

To add modules we use:
```
    $ sudo yarn add <packagename>
```
Adding with dev dependiences:
```
    $ sudo yarn add -D <packagename>
```
<br>

# How to RUN

To run frontend
```
    $ sudo yarn start
```
